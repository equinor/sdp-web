Frequently Asked Questions

Can you add xxx linux package to /prog/sdpsoft?

Yes, we can. But it requires some work and will take some time. It is often not prioritized unless everyone in Statoil realy need that package right now.
We therefor recommend everyone to utilize pythons "Virtual Environments" functionality. This enables you to install any python package you'd like and have all the necessary packages be linked to the python project in an efficient way.
Alternatively run pip install with the "--user" option to install the packages in your home directory.
Or, even better, get a Linux laptop or Linux virtual machine and have no limitations!
