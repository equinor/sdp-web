Frequently Asked Questions

Can you add this package to SDPSoft?

If it makes sense, yes, it's all dependent on what you need. If your software is going to be added, know that compiling software can be tedious and troublesome and time consuming, so don't expect it to be available immediately. We often end up in a dependency hell.

Software that will benefit alot of users will be prioritized.

Can you add this python module to SDPSoft?

If it makes sense, yes. We strongly recommend considering if python virtual environments can solve your problem instead. You could for instance use one of the python versions in SDPSoft to create your virtual environemtn and from there only use that virtual environment and install whichever module you want.

Can you add this plugin to Jenkins?

If you have your own Jenkins instance, either you install it yourself, or you ask the Jenkins master admin (usually the one responsible for the project the instance is for).

Installing plugins is easy: Go to Manage Jenkins --> Manage Plugins --> Available tab --> Filter/Search --> Tick checkbox(es) --> Install

If its a managed Jenkins instance, we can do this for you. However, we might have to restart the Jenkins master after installation or the plugin might be enabled right away. If the master needs to be restarted, know that we will wait until the master isn't doing any builds and all further builds will be freezed (maintenance mode). In some circumstances, this waiting can take a while.
