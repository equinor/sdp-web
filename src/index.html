Software Development Platform Team (SDPTeam)

Our goal is to provide software development tools and support, as well as other aspects of maintaining and developing better code and software for Statoil.

Beneath you'll find the tools and services we provide. Missing something from the list? Don't hesitate to ask us for tools and/or services you think should be part of our portfolio.

Wikidocs https://sdp.statoil.no/wikidocs

Easy and fast file-sharing filesystem only accessible within the Statoil network. It's typically used to host content (PDF, images, documents, videos) for the Statoil Wiki. See this wiki page for more information

SDP Soft

A collection of compiled software that are mounted on a shared NSF at different Statoil premises in Norway and around the world. Access it on /data/sdpsoft

Github https://github.com/statoil

Statoil's playground at Github, a cloud service offering version control repositories.

Gitlab https://git.statoil.no

Distributed version control designed to handle everything from small to very large projects with speed and efficiency. Read more about it on the Statoil wiki:

  • Software:Git
  • Software:Git_Tutorial
  • Software:User_Gitlab

Jenkins

Tronheim => ci.statoil.no/tr

Bergen => ci.statoil.no/be

Stavanger => ci.statoil.no/st

The rest => ci.statoil.no/others

Easy-to-use continuous integration (CI) system for increased productivity. Choose the Jenkins instance that feels appropriate for your team/project.

Jira

statoil.atlassian.net

Jira is a widely acknowledged task management system with a lot of other goodies underneath the hood. Need access to an internal Jira instance? Contact us.
