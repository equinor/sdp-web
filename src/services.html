Services

Statoil Wiki http://wiki.statoil.no

A wiki using the same software as Wikipedia to provide a true wiki experience for projects in Statoil.

Wikidocs https://sdp.statoil.no/wikidocs

Easy and fast file-sharing filesystem only accessible within the Statoil network. It's typically used to host content (PDF, images, documents, videos) for the Statoil Wiki. See this wiki page for more information

Developer blog http://developer.statoil.no

Opinions, discussions and information from the programmer community at Statoil

SDP Soft

A collection of compiled software that are mounted on a shared NSF at different Statoil premises in Norway and around the world. Access it on /data/sdpsoft

Github https://github.com/statoil

Statoil's playground at Github, a cloud service offering version control repositories.

Gitlab https://git.statoil.no

Distributed version control designed to handle everything from small to very large projects with speed and efficiency. Read more about it on the Statoil wiki:

  • Software:Git
  • Software:Git_Tutorial
  • Software:User_Gitlab

Subversion http://svn.statoil.no

Simple and centralized version control. Read more about it on the Statoil wiki Subversion

Jenkins http://ci.statoil.no

Easy-to-use continuous integration (CI) system for increased productivity.

Jira https://sdp.statoil.no/jira

Jira is a widely acknowledged task management system with a lot of other goodies underneath the hood.

Jira HPC https://sdp.statoil.no/jira_hpc

Jira is a widely acknowledged task management system with a lot of other goodies underneath the hood. NB! This is JIRA for HPC.

Trac https://sdp.statoil.no/trac

Trac is an enhanced wiki and issue tracking system with good subversion integration for software development projects

Docker Registry http://git-registry.statoil.no

Docker Registry is a stateless, highly scalable server side application that stores and lets you distribute Docker images. The Registry is open-source, under the permissive Apache license
