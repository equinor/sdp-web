GitHub

In order to apply for access to GitHub you need to have a user on github.com which has 2FA enabled.

Requests for access in AccessIT should then include your pre-existing github.com user in the Statoil organization on github.com. If you'd like to get access to an existing Statoil repository or become member of an existing Statoil team, you need an admin for the repo or team to invite you.

Make sure you fill out your public profile (name, email, company and location) under user menu settings.
